🚀  Color Tracking Cod |  تتبع الألوان كود

A high-performance real-time color tracking system designed for precision and speed.
نظام متقدم لتتبع الألوان بشكل لحظي، مصمم ليقدم دقة عالية وأداء سريع في تحليل الشاشة.

This project focuses on detecting and tracking specific colors dynamically using optimized screen analysis techniques.
يركز المشروع على اكتشاف وتتبع ألوان محددة بشكل ديناميكي باستخدام تقنيات تحليل شاشة محسّنة.
